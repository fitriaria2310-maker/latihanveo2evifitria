# latihanveo2evifitria

Proyek sederhana: situs statis untuk penjualan kitab pondok pesantren.

File penting:

- [index.html](index.html)
- [styles.css](styles.css)
- [script.js](script.js)

Menjalankan secara lokal:

```bash
# Buka langsung file `index.html` di browser, atau jalankan server HTTP sederhana:
python3 -m http.server 8000
# lalu buka http://localhost:8000
```

Jika ingin fitur tambahan (checkout nyata, backend, atau desain lain), beri tahu saya.
